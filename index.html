first, import the super-rect styles in any css file 


@import 'https://www.dommandy.com/super-rect/squircle.css';

add the rounded class to any html element 


<div class="rounded"></div>

add the bordered class to add a super-rect border 


<div class="rounded bordered"></div>

add any of --border-radius, --border-size, --border-color, or --border-background to your css to customize 


.your-class {
    --border-radius: 3em;
    --border-size: 10px;
    --border-color: #333;
    --border-background: #c00;
}

here's that custom styling applied 


<div class="rounded bordered your-class"></div>
